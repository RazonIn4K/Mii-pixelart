# Reference Pack Template

This directory contains the template for exported reference packs. When a user clicks "Export Reference Pack" in the studio, the following files are generated:

## Contents

| File | Format | Description |
|------|--------|-------------|
| `{project-name}-{timestamp}.json` | JSON | The complete GridDocument project file |
| `{project-name}-guide.png` | PNG | Labeled pixel guide with grid lines and paint-by-numbers |
| `{project-name}-reference.html` | HTML | Self-contained reference page with palette and project data |

## How to Use the Reference Pack

1. **Open the HTML reference page** in any browser for a quick overview of the project, palette, and grid data.
2. **Open the PNG guide** on a second screen or print it out. Each cell has a number corresponding to a color in the palette.
3. **Open Tomodachi Life** and navigate to the Palette House.
4. **Follow the guide square by square**, using the color numbers to find the right swatch in the game's palette.
5. **Save the JSON file** if you want to re-open the project in the studio later for further editing.
