# Fixtures

This directory contains test fixtures for the Living The Grid Repaint Studio.

## Files

| File | Description |
|------|-------------|
| `sample-grid-document.json` | A minimal 4×4 test grid using 4 Tomodachi Life palette colors |
| `living-the-grid-*.json` | **Place the real Living The Grid JSON export here** |

## Adding the Real Fixture

To complete Phase 0 of the roadmap:

1. Open [livingthegrid.com](https://livingthegrid.com/) in your browser.
2. Upload any image and convert it.
3. Click the save/export button to download the JSON file.
4. Place the downloaded file (e.g., `living-the-grid-1777253291337.json`) in this directory.
5. Follow the steps in `docs/json-schema-discovery.md` to inspect and document the format.
